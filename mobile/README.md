# AuraTalk (Flutter)

This Flutter app is scaffolded to match the provided AuraTalk reference screens (light theme, indigo primary, rounded cards, center-highlight bottom nav).

## Run
```bash
cd mobile
flutter pub get
flutter run
```

## UI Notes
- Tabs: Home / Zone / Chats / Calls / Profile
- Current focus: theme + layout skeleton only (no backend auth wired yet)
